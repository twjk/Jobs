# Android-xiaozhi文档目录

本目录包含了 Android-xiaozhi 项目 的全部功能文档，按功能模块进行划分，便于查阅和使用。

项目基于Flutter框架开发，支持iOS、Android、Web等多平台，提供丰富的语音交互和AI对话功能。
视觉识别功能需配置API Key才可使用。
